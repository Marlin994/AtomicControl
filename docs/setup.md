# Setup

## Requirements

- Minecraft with CC:Tweaked
- Extreme Reactors / Bigger Reactors compatible computer ports
- Advanced Monitor recommended
- Wired modem network recommended

## Installation

Copy all files from `src/` onto your ComputerCraft computer.

Start:

```lua
main
```

Run setup again:

```lua
main setup
```

## Language

After first start, edit:

```lua
reactor_turbine_controller.cfg
```

Set:

```lua
language = "de"
```

or

```lua
language = "en"
```
